About:

  • Using the text inputs enter the coordinates of two points given on the Squad map. This calculator will output the Distance, Milliradians, and Bearing. The default format for the coordinates is 'Short Format 1' which means if you see 
    A3-KP2-1
    in the map, you would input 
    A321
    or 
    A3K11
    . You can change the input format by using the selector.
  • To minimize the error enter all of the data given in the Squad map. The min / max values are calculated from the error. The error is based on the number of keypad values you input. If you input all of the data given in the Squad map then the maximum error will be a distance of 47.1 meters in the worst case, which will affect the millradian min / max values
  • The error is a 'worst case' approximation. To minimize error, center your mouse so that it is at the center of a keypad (in other words, don't put your mouse on the gridlines as this will increase the error).
  • This site is mobile friendly and the distance is calculated using JavaScript entirely in your browser. The best way to use this calculator is with your mobile phone as you play the game.
  • Case insensitive: 
    A5-KP2-3
    is the same as 
    a5-kp2-3
    .
  • There are 3 short formats to make it so you don't have to write so much.
  • With the Long Format and Short Format 1, the 
    P
    is optional: 
    A5-K2-3
    is the same as 
    A5-KP2-3
    . With Short Format 1, the 
    K
    is optional, 
    A4K32
    is the same as 
    A432
    .
  • The distance, bearing and milliradians are calculated from the distance between the top-left-corner of the two grids. This may affect how you want to dial in your shots. See the image to the right: Your target can be anywhere within the red square, and your mortars can be anywhere in the green square.
